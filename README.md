 Автоматизована Система Доставки (Delivery Accounting System)
Проєкт розроблено в рамках лабораторної роботи. Це повноцінний застосунок для управління процесами доставки їжі з розділенням ролей (Клієнт та Кур'єр).
 Технології та Архітектура
Мова: C# (.NET 10)
Інтерфейс: Windows Forms + Console Mode
Збереження даних:JSON 
Патерни: Separation of Concerns (чітке розділення UI та бізнес-логіки)

Головні фішки проєкту
1.Кастомний Flat-дизайн:** Повна відмова від застарілих компонентів (TabControl) на користь сучасних панелей.
2. Повноцінна Темна тема (Dark Mode): Реалізована через рекурсивний обхід та динамічне перефарбовування всіх елементів форми.
3. Smart-Кошик:** Автоматичний підрахунок суми, підтримка промокодів (напр. `HALYAVA`) та генерація замовлень.
4. Живий пошук (Live Search): Миттєва фільтрація каталогу товарів за допомогою технології **LINQ**.
5. Гібридний запуск: Можливість вибору режиму роботи при старті (Сучасний GUI або класична Консоль).
6. Безпека: Файли баз даних з паролями (`users.json`) додані в `.gitignore` та не відслідковуються в публічному репозиторії.

## 👥 Ролі в системі
* **Клієнт:** Може переглядати меню, шукати товари, збирати кошик, застосовувати купони та переглядати історію своїх замовлень.
* **Кур'єр:** Має власне робоче місце для прийняття нових замовлень в роботу, зміни їх статусів ("В дорозі", "Доставлено") та відстеження своєї зайнятості.
